# Chatobby 0.1.0 public alpha

Initial Community plugin candidate for Chatobby's desktop Obsidian connector.

## Included

- Obsidian-native conversation, session, memory, permission, event, query,
  channel, and subagent interfaces
- Authenticated local connection to the separately installed Chatobby runtime
- Signed-runtime discovery, lifecycle recovery, and first-run download guidance
- Provider, model, effort, and session permission controls
- Image and document attachments plus structured tool and task presentation

## Install

Install the connector through Obsidian, then use **Get runtime** to download the
separate Windows runtime from the official
[`chatobby-runtime`](https://github.com/TitanicEclair/chatobby-runtime/releases/latest)
release.

This is free alpha software. Back up important vaults and start with a copied
test note. Optional Patreon support does not unlock features.
